# Result Copy

This document locks the current direction for the top narrow screen's result text.

It does not define the final product name, the final screen name, or the names of the three reading layers. It only defines what the small top screen should say at the result moment.

The top narrow screen should be treated as the result title layer. It is not a clock, date, or generic status label.

## Core Feeling

The top screen should feel like a short page from an answer book, but it must not behave like fortune-telling.

It is closer to:

- A small sign on the road.
- A sentence found during a journey.
- A calm witness that speaks once.
- A two-line verdict that returns agency to the user.

It should not feel like:

- AI advice.
- Therapy copy.
- Productivity coaching.
- A fortune slip.
- A system command.

## Copy Shape

Default shape:

```text
Observation.
Agency turn.
```

The first line says what the device saw.

The second line refuses to turn that signal into permission, command, or destiny.

Examples:

```text
三盏灯看向同处。
这不是许可。
```

```text
答案太整齐了。
它们都同意，但你还在。
```

Short stamp-like lines are allowed, but they should be rare:

```text
惯性占优
```

```text
NOT FIXED
```

## Writing Rules

- Usually two lines.
- One line can be very short.
- Prefer concrete images: light, road, signal, echo, answer, direction.
- Use "system" sparingly.
- Do not explain the three readings.
- Do not say good, bad, correct, wrong, safe, or dangerous.
- Do not tell the user what to do.
- Allow a little story, but keep it sharp.
- Let the final line return agency.

## Canonical Candidates

These are the strongest current candidates and should be used first in UI prototypes.

### Unanimous

```text
太一致了。
但一致不等于答案。
```

```text
三盏灯看向同处。
这不是许可。
```

```text
这次没有异声。
系统合上了，你没有。
```

```text
答案太整齐了。
它们都同意，但你还在。
```

```text
没有反对声。
但这不等于通过。
```

```text
路面很平。
脚还在你这里。
```

### Majority Maintain

```text
惯性占优
```

```text
熟路先亮了。
但熟悉不是答案。
```

```text
旧答案回来了。
它很会发光。
```

```text
你正在靠近原路。
先别把省力当作同意。
```

```text
多数选择维持。
NOT FIXED.
```

```text
系统看见了惯性。
你还可以不照做。
```

### Minority Deviate

```text
有个答案站了起来。
偏离不是错误。
```

```text
路的另一边有信号。
它还没有熄灭。
```

```text
不是所有灯都看向同处。
你身上仍有回声。
```

```text
有一盏没有归队。
先别急着按下它。
```

```text
小路亮了一下。
不代表你要走，但它在。
```

```text
另一种你没有消失。
它只是比较小声。
```

### Static Or Unclear

```text
信号没有站稳。
这一轮先别相信。
```

```text
这页被风吹乱了。
稍后再看。
```

```text
灯还没形成方向。
不要急着读它。
```

```text
声音没有落地。
再靠近一点。
```

```text
这一轮太短。
答案还没醒。
```

## Strong Single-Line Stamps

Use these as occasional result strips, loading endings, or debug/demo states.

- 惯性占优
- NOT FIXED
- 没有异声
- 小路亮了
- 系统合上了
- 另一边有信号
- 这不是许可
- 偏离不是错误

## Product Reading

The result copy should support the core product position:

The system can reveal a signal, but it cannot turn that signal into permission.

The user is still present.
